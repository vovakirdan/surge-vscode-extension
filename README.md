# Surge Syntax Highlighting for VS Code

Это расширение VS Code добавляет подсветку синтаксиса и базовый анализ кода для файлов Surge (`.sg`).

## Возможности

- Распознаёт ключевые слова, типы, литералы и новые конструкции (`extern`, `alias`, `literal`, `using`).
- Обеспечивает собственные цвета подсветки только для языка Surge, не меняя активную тему и не влияя на другие языки.
- Поддерживает оформление комментариев, строк, чисел и аннотаций (`@pure`, `@override`).
- Выполняет диагностику и семантическую проверку через установленный `surge` (`surge diag --format json`) и отображает ошибки прямо в редакторе (если `surge` недоступен, расширение продолжит работать только как подсветка синтаксиса).

## Установка из исходников

1. Скопируйте папку `vscode-extension` в отдельное место или откройте репозиторий в VS Code.
2. Выполните команду `npm install -g vsce` (при необходимости) и упакуйте расширение: `vsce package`.
3. Установите полученный `.vsix` через команду `Extensions: Install from VSIX...` в VS Code.

Либо используйте запуск из исходников: нажмите `F5` в VS Code, чтобы открыть новое окно с активированным расширением.

## Папки и файлы

- `package.json` — описание расширения и правила подсветки цветов.
- `extension.js` — интеграция с `surge diag --format json` при изменении файлов и публикация ошибок в VS Code.
- `language-configuration.json` — конфигурация редактора для комментариев и пар скобок.
- `syntaxes/surge.tmLanguage.json` — грамматика TextMate для языка Surge.

## Настройка

- `surge.analyzer.executablePath` — путь до исполняемого файла `surge`, используемого для запуска `surge diag`. По умолчанию ожидается, что `surge` доступен в `PATH`.
